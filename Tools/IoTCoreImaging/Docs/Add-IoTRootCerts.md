# Add-IoTRootCerts

## SYNOPSIS
Generates the RootCerts package (Security.RootCerts) contents based on the workspace specifications.

## SYNTAX

```
Add-IoTRootCerts
```

## DESCRIPTION
Generates the RootCerts package (Security.RootCerts) contents based on the workspace specifications.
You will need to import the required certificates into the workspace before using this command.

### EXAMPLE 1
```
Add-IoTRootCerts
```
