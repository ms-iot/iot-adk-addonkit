# Import-PSCoreRelease

## SYNOPSIS
Import Powershell Core Release into your workspace and update the wm xml files.

## SYNTAX

```
Import-PSCoreRelease [[-Version] <String>] [<CommonParameters>]
```

## DESCRIPTION
Import Powershell Core Release into your workspace and update the wm xml files.

## EXAMPLES

### EXAMPLE 1
```
Import-PSCoreRelease  7.0.0
```
